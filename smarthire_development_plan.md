# SmartHire 招聘与面试管理系统开发计划

## 1. 项目简介

**SmartHire** 是一个基于 Java 后端的招聘与面试管理系统，目标是模拟真实企业招聘平台的核心业务流程，支持岗位发布、简历投递、面试安排、通知提醒和数据统计。

本项目的主要目的不是做一个“能跑的课程作业”，而是做一个：

- 可以写进简历的 Java 项目
- 可以在面试中完整讲清楚的后端项目
- 能体现工程能力、业务建模能力和技术栈深度的项目

---

## 2. 项目目标

通过本项目，重点展示以下能力：

- Java 后端开发能力
- Spring Boot 项目结构设计能力
- MySQL 数据库建模与接口设计能力
- JWT 认证与权限控制能力
- Redis 缓存与 RabbitMQ 异步处理能力
- Docker 部署与工程化能力
- 基本前后端协作能力（React）

### 4.2 HR

- 更新申请状态
- 安排面试
- 查看招聘统计

### 4.3 管理员（Admin）
- 用户管理
- 角色管理
- 系统统计
- 操作日志查看

## 5. 核心功能模块

### 5.1 用户与认证模块
- 用户注册
- 用户登录
- JWT 鉴权
- 密码加密存储
- 角色权限控制（RBAC）

### 5.2 岗位管理模块
- HR 发布岗位
- 岗位详情查看
- 岗位更新与删除
- 岗位状态管理（上线 / 下线 / 过期）
- 岗位分页与条件搜索

### 5.3 简历投递模块
- 候选人投递岗位
- 上传简历 PDF
- 查看个人投递记录
- HR 查看岗位对应投递列表
- 防重复投递逻辑

### 5.4 面试管理模块
- HR 安排面试
- 更新面试时间、地点、备注
- 候选人查看面试安排
- 更新面试结果

- 投递成功通知
- 面试安排通知
- 状态更新通知

- 候选人状态分布统计

### 5.8 运维与部署模块（增强）
- Dockerfile

## 11. 开发原则

本项目遵循以下原则：

1. **先做业务闭环，再做技术增强**
2. **先保证后端完整，再补前端体验**
3. **优先做面试可讲的功能，而不是花哨页面**
4. **所有新增功能都尽量服务于“简历可写 + 面试可讲”**
